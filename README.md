# leify

`leify` leifies your sentences!

## Usage

```
usage: leify [-h] text

positional arguments:
  text        the text to be leified

optional arguments:
  -h, --help  show this help message and exit
```

## Examples

```
python leify.py
    Leify what? Hello World!
    le Hello le World!
```

```
python leify.py
    Leify what? Your mum a hue.
    Le Your le mum le a le hue.
```

Happy leifying! :^)
